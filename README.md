## Smart Social Distancing Belt
In the midst of the chaos of the Covid-19 virus, keeping distance is something that can be done to avoid transmission of this virus. We cannot recklessly social contact with anyone.
This tool is useful for keeping a distance from someone. If someone approaches, the alarm will ring to warn the person not to approach.
